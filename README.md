# Cadastro de Pessoas
Mini-projeto de manipulacao de dados no qual eu coloco em pratica o que estou aprenendo no curso de Java.



## Sobre
Este mini projeto é apenas um sistema simples de CRUD no qual é possivel adicionar, listar, remover e atualizar dados de uma pessoa.
Os dados sao salvos em memoria e nao em banco de dados ou arquivos exernos.

## Funcionalidades 

- Cadastro de uma pessoa passando nome, idade e email.
- Alteração de dados da pessoa ja cadastrada.
- Listagem de pessoas ja cadastradas.
- Remoção de pessoa cadastrada por meio do id.
